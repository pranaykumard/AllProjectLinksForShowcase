Target Detection and Localization using Deep Learning
    Project aimed at developing deep learning approach to localise targets in videos which are captured from a
    UAV(unmanned aerial vehicle) using corresponding satellite images of that region.
    Its based on detecting objects in a video and making a bounding box around it. We have used openCV, darkf low
    tools which are YOLO v2 model, convolutional neural net works which helped us in achieving what we needed.
    We could able to achieve 77% accuracy to find objects in untrained images.
    https://github.com/pranaykumard/TargetDetectionAndLocalization has further details about project

Ecommerce Project using spring boot

    and Kafka, covering functionalities from user authorization to payment gateway integration,tested with JUnit and
    deployed on AWS Elastic Beanstalk leveraging RDS for Data Persistence, CloudWatch for efficient Logging and
    Monitoring.
    Implemented powerful sorting, filtering and paging to allow for efficient discovery of products.
